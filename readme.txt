Angular adalah framework javascript yang memungkinkan untuk membuat Single-Page-Applications (SPAs) reaktif. kerangka pemrograman struktural untuk mendapatkan web app yang dinamis.
Hal ini memungkinkan menggunakan HTML sebagai bahasa template serta memperluas sintaks HTML sehingga seluruh komponen dalam aplikasi yang tengah dibangun bisa diekspresikan secara jelas dan ringkas.
angular hanya menggunakan satu file HTML dan beberapa Typescript. 

- module
    module adalah gabungan dari beberapa fitur/component yang terhubung

- component
    component adalah suatu fungsi yang akan menyusun tampilan web dan fitur-fitur dalam web itu

Hubungan antara component dan module adalah Modulediisi oleh kelompok component yang saling berhubungan. 
sementara component membutuhkan module untuk mengidentifikasi component lain yang memiliki hubungan yang sama.
